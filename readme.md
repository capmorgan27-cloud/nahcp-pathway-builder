# NAHCP Career Pathway Builder™

This repository contains Version 1.0 of the enterprise Career Pathway Builder application for the National Academy of Healthcare Professions (NAHCP). It guides students seamlessly from 10-hour Essentials up through the 220-hour Executive Leadership Fast Track™.

## Tech Stack
* **Frontend/Backend:** Python, Streamlit
* **Database:** SQLite
* **Data Manipulation & Visualization:** Pandas, Plotly, Matplotlib
